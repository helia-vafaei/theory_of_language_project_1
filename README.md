# theory_of_language_project_1
# input
You will be given a Finite Automata.
# output
If the input string is accepted in the Finite Automaton machine, the word Accepted is printed, otherwise the word Rejected is printed.
# example
input:
<br>
{A,B}
<br>
{a,b}
<br>
{B}
<br>
4
<br>
A,b,A
<br>
A,a,B
<br>
B,a,B
<br>
B,b,B
<br>
bbb
<br>
output:
<br>
Rejected
